
## Features

- Easy setup with just `npm install` and `npm start`.
- Runs locally on your machine.
- Built with Node.js and JavaScript.

## Getting Started

Follow these instructions to get a copy of the project running locally.

## Prerequisites

- [Node.js](https://nodejs.org/) installed
- [npm](https://www.npmjs.com/) (comes with Node.js)

### How to run it locally

1. Clone the repository:

git clone https://github.com/peyn-dev/forkcast.git

2. cd forkcast

3. npm install

4. npm start

5. Open your browser or click the link in your terminal: http://localhost:3002
